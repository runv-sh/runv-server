Junte-se

Uma chave SSH no seu computador e um primeiro contacto com entre@runv.club — em qualquer sistema.

Pedido de entrada entre@runv.club

Em resumo

  1. Gere um par de chaves SSH no seu PC (fica só a pública para colar no pedido).
  2. Ligue com ssh a entre@runv.club e siga o que aparecer no ecrã.
  3. Aguarde — a equipa revê a fila; não é cadastro instantâneo.

Linux

Terminal: gere Ed25519 e copie a linha da chave pública (.pub).

ssh-keygen -t ed25519 -C "seu-email-ou-pc" -f ~/.ssh/id_ed25519_runv
cat ~/.ssh/id_ed25519_runv.pub

macOS

Terminal (Spotlight: “Terminal”) — os mesmos comandos que em Linux. OpenSSH já vem no sistema; opcional: ssh-add ~/.ssh/id_ed25519_runv.

ssh-keygen -t ed25519 -C "seu-email-ou-pc" -f ~/.ssh/id_ed25519_runv
cat ~/.ssh/id_ed25519_runv.pub

Windows

PowerShell ou Terminal do Windows. Confirme o cliente: ssh -V. Se faltar, instale Cliente OpenSSH em Configurações → Aplicativos → Recursos opcionais.

ssh-keygen -t ed25519 -C "seu-email-ou-pc" -f $env:USERPROFILE\.ssh\id_ed25519_runv
Get-Content $env:USERPROFILE\.ssh\id_ed25519_runv.pub

Nunca partilhe o ficheiro sem .pub — esse é o privado.

Ligar ao entre@runv.club

Linux / macOS:

ssh -i ~/.ssh/id_ed25519_runv entre@runv.club

Windows (PowerShell):

ssh -i $env:USERPROFILE\.ssh\id_ed25519_runv entre@runv.club

Na primeira vez, aceite a fingerprint se confiar no servidor. Se a sua chave tiver outro nome ou estiver no caminho por defeito, pode omitir -i.

Dúvidas ou bloqueios: admin@runv.club. Tipos de chave: preferência por Ed25519; também ECDSA ou RSA conforme política do servidor.
