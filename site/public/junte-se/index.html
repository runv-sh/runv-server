Junte-se

Uma chave SSH no seu computador e um primeiro contato com entre@runv.club — de um jeito simples, em qualquer sistema.

Pedido de entrada entre@runv.club

O que você precisa

Um computador e alguns minutos

Você só precisa gerar uma chave SSH, copiar a parte pública e seguir o fluxo guiado do entre@runv.club.

Sem pressa

Não precisa saber tudo agora

Se esta for sua primeira vez com SSH, tudo bem. A ideia aqui é justamente tornar esse começo mais simples e amigável.

Em resumo

  1. Gere um par de chaves SSH no seu computador. Você vai usar só a chave pública no pedido.
  2. Conecte-se com ssh a entre@runv.club e siga o que aparecer na tela.
  3. Aguarde a revisão. O processo é manual e não há cadastro instantâneo.

Se esta é a sua primeira vez usando SSH, tudo bem: esta página foi pensada justamente para ajudar você a dar esse primeiro passo com calma.

Linux

No terminal, gere uma chave Ed25519 e copie a linha da chave pública (.pub).

ssh-keygen -t ed25519 -C "seu-email-ou-pc" -f ~/.ssh/id_ed25519_runv
cat ~/.ssh/id_ed25519_runv.pub

macOS

No Terminal (Spotlight: “Terminal”), use os mesmos comandos do Linux. O OpenSSH já vem no sistema; se quiser, pode adicionar a chave com ssh-add ~/.ssh/id_ed25519_runv.

ssh-keygen -t ed25519 -C "seu-email-ou-pc" -f ~/.ssh/id_ed25519_runv
cat ~/.ssh/id_ed25519_runv.pub

Windows

No PowerShell ou Terminal do Windows, confirme primeiro se o cliente existe com ssh -V. Se faltar, instale o Cliente OpenSSH em Configurações → Aplicativos → Recursos opcionais.

ssh-keygen -t ed25519 -C "seu-email-ou-pc" -f $env:USERPROFILE\.ssh\id_ed25519_runv
Get-Content $env:USERPROFILE\.ssh\id_ed25519_runv.pub

Nunca compartilhe o arquivo sem .pub. Esse é o arquivo privado e deve ficar só com você.

Ligar ao entre@runv.club

Linux / macOS:

ssh -i ~/.ssh/id_ed25519_runv entre@runv.club

Windows (PowerShell):

ssh -i $env:USERPROFILE\.ssh\id_ed25519_runv entre@runv.club

Na primeira vez, aceite a fingerprint se estiver conectando ao servidor certo. Se a sua chave tiver outro nome ou estiver no caminho padrão, você pode omitir -i.

Dúvidas ou bloqueios: admin@runv.club. A preferência é por chaves Ed25519; ECDSA e RSA também podem ser aceitas conforme a política do servidor.

Depois que a conta for aprovada, você poderá entrar na pubnix, criar sua página pessoal, explorar o ambiente e conversar com a comunidade no IRC.
